Giphy

With this API, you can search for any gifs that your heart desires! I set a limit to how many gifs you will receive, which is 10 to allow for fast page loading.

These gifs will fill into the Bootstrap container that I set.

Open Weather Map

Using Open Weather's API, you can type in your zip code and search for the current conditions in your area. This is currently only searching US locations only.

The weather temperature and condition append a div container using AJAX, with Bootstrap styling.
